Language Basics

Variables and Data Types

Python is dynamically typed, meaning you don't need to specify the variable's type explicitly. Common data types include integers, floats, strings, lists, tuples, dictionaries, and sets.

            # Example of variable declaration and assignment
              name = "John Doe"
              age = 30
              is_student = True

Operators

Python supports various operators, including arithmetic, comparison, logical, and assignment operators.

            # Arithmetic Operators
            x = 10
            y = 3
            result = x + y
            
            # Comparison Operators
            is_equal = x == y
            
            # Logical Operators
            is_true = True and False
            
            # Assignment Operators
            x += 1  # Equivalent to x = x + 1

Conditional Statements

Python uses if, elif, and else statements for conditional branching.

            # Example of if-elif-else statement
            x = 10
            if x > 10:
                print("x is greater than 10")
            elif x == 10:
                print("x is equal to 10")
            else:
                print("x is less than 10")

Loops

Python provides for loops and while loops for iteration.

            # Example of a for loop
            fruits = ["apple", "banana", "orange"]
            for fruit in fruits:
                print(fruit)
            
            # Example of a while loop
            count = 0
            while count < 5:
                print(count)
                count += 1

Functions

Functions in Python are defined using the def keyword.

            # Example of a function
            def greet(name):
                return "Hello, " + name + "!"
            
            message = greet("Alice")
            print(message)  # Output: Hello, Alice!

Classes and Objects

Python supports object-oriented programming, and classes can be defined using the class keyword.

            # Example of a class
            class Rectangle:
                def __init__(self, width, height):
                    self.width = width
                    self.height = height
            
                def area(self):
                    return self.width * self.height
            
            # Creating an object of the Rectangle class
            rect = Rectangle(5, 10)
            print(rect.area())  # Output: 50

Modules and Packages

Python code can be organized into modules, and multiple modules can be grouped into packages. Modules are imported using the import statement.

            # Example of importing a module and using a function from it
            import math
            
            print(math.sqrt(25))  # Output: 5.0
Advanced Concepts

Exception Handling

Python allows you to handle exceptions using try, except blocks.

            # Example of exception handling
            try:
                result = 10 / 0
            except ZeroDivisionError:
                print("Error: Division by zero.")

File Handling

Python provides functions to work with files, like opening, reading, and writing.

            # Example of file handling
            with open("example.txt", "r") as file:
                content = file.read()
            
            print(content)

Decorators

Decorators are used to modify the behavior of functions or methods.

            # Example of a decorator
            def my_decorator(func):
                def wrapper():
                    print("Something is happening before the function is called.")
                    func()
                    print("Something is happening after the function is called.")
            
                return wrapper
            
            @my_decorator
            def say_hello():
                print("Hello!")
            
            say_hello()

Generators

Generators allow you to create iterators using the yield statement.

            # Example of a generator
            def countdown(n):
                while n > 0:
                    yield n
                    n -= 1
            
            for num in countdown(5):
                print(num)

List Comprehensions

List comprehensions provide a concise way to create lists.

            # Example of a list comprehension
            squares = [x * x for x in range(10)]
            print(squares)  # Output: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

Regular Expressions

Python supports regular expressions for pattern matching and manipulation.

            # Example of regular expression
            import re
            
            pattern = r"\d+"
            text = "Hello 123 World"
            matches = re.findall(pattern, text)
            print(matches)  # Output: ['123']

Multithreading and Multiprocessing

Python allows multithreading and multiprocessing for concurrent execution.

            # Example of multithreading
            import threading
            
            def print_numbers():
                for i in range(1, 6):
                    print(i)
            
            thread = threading.Thread(target=print_numbers)
            thread.start()

Web Scraping

Python can be used for web scraping using libraries like BeautifulSoup and Requests.

            # Example of web scraping with BeautifulSoup and Requests
            import requests
            from bs4 import BeautifulSoup
            
            url = "https://example.com"
            response = requests.get(url)
            
            soup = BeautifulSoup(response.content, "html.parser")
            title = soup.title.get_text()
            print(title)
